Integration & Endpoints

SecureHide provides a RESTful API for calculating image capacities programmatically.

Calculate Image Capacity

Returns the maximum character capacity of an image.

POST /api/capacity/
Parameters:
Parameter Type Description
image File (multipart/form-data) The PNG or BMP image file.
Response:
{
  "success": true,
  "capacity": {
    "total_pixels": 1048576,
    "max_bytes": 393216,
    "max_chars": 393216
  }
}
